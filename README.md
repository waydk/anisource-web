## Development
### System dependencies
* Python 3.8
* Flask 2.0.1
* PicImageSearch 2.1.1
* environs 9.3.3
### Setup environment
* Create .env and set [SAUCE_API](https://saucenao.com/user.php?page=search-api)
### Launch
* <code>python3 run.py</code>
